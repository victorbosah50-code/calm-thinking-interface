# Calm Thinking Interface

A minimalist, public-good web tool designed to help people think clearly and calmly.

## What This Is
- A slow, structured reflection space
- No tracking
- No persuasion
- No optimization
- No data storage

## What This Is Not
- Not therapy
- Not medical advice
- Not coaching
- Not AI decision-making

## Ethics
This project prioritizes human agency, calm cognition, and reversibility.
